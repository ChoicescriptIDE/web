Command: author

Used in startup.txt immediately following *title, *author specifies the author's name to be displayed within the game below the game title.

Example Usage

*title The Most Awesome Game Ever!
*author Lord Sauron
*scene_list
	startup
	somescene
	anotherscene
	yetanotherscene
	ohgodwilliteverend
